How should the AI get smart enough to do this?

The current system sends this week's numbers. For signature move + best night, it needs more. Pick your approach.

A

Rewrite prompts only

Change the tone and structure of existing prompts. Faster voice, friend tone, no lectures. But: signature move and best night need historical data the current system doesn't pass — this option skips those for now.

Pros

  • Ship in one session
  • Zero backend changes

Cons

  • No signature move
  • No real best night
B

Pass more history to the AI — let it figure out patterns

iOS app sends last 8–12 weeks of night summaries with the monthly prompt. Claude reads the history and surfaces the signature move + best night itself.

Pros

  • Signature move works
  • Best night works
  • Minimal new backend code

Cons

  • Bigger prompt = more tokens
  • AI may still miss subtle patterns
C

Pre-compute insights in code, AI just narrates

iOS or Firebase function finds the best night, detects the pattern (e.g. "always accelerates after midnight"), then hands those facts to Claude to write naturally. AI never has to "figure it out" — it just makes it sound good.

Pros

  • Most reliable insights
  • AI output is sharper
  • Cheaper tokens

Cons

  • More Swift/JS code to write
  • More planning

My recommendation: C — code finds the facts, AI tells the story. That's the cleanest split and gives the sharpest output.
